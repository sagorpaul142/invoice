# Invioce

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Navigate server folder then json server run
```
cd server
yarn install
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
